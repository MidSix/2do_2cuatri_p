# 001 - Manual de Prácticas de Computación Concurrente, Paralela y Distribuida

Este documento describe la estructura y el estilo de los manuales de prácticas para la asignatura de Computación Concurrente, Paralela y Distribuida.

## Puntos Clave:

*   **Contexto:** Cada práctica incluirá una introducción que la contextualizará con la teoría o con prácticas anteriores. Las prácticas son auto-contenidas y auto-explicativas.
*   **Instrucciones Previas ("Antes de empezar"):** Se destacarán instrucciones críticas o dependencias necesarias antes de iniciar la práctica, cruciales para evitar problemas.
*   **Objetivos:** Se listarán los conceptos clave y técnicas de paralelismo a comprender.
*   **Índices:** Después del contexto y los objetivos, se proporcionarán índices para la estructura del documento, tareas y figuras.
*   **Estructura y Estilo:** Los documentos se organizarán en secciones y subsecciones, explicando los conceptos de forma guiada.
    *   **Tareas:** Se incluirán tareas a resolver, algunas breves y otras de desarrollo libre. Algunas podrían ser reutilizadas para la evaluación.
    *   **Notas/Anotaciones:** Se usarán "apuntes" o "notas" informativas, aclaratorias o de advertencia, marcadas con símbolos distintivos (ej. `i` para información, `!` para atención) y claramente separadas del texto principal.
    *   **Código:** Las referencias a código se harán mediante la mención explícita del fichero o mediante cuadros de estilo con números de línea. Se recomienda usar los ficheros de código proporcionados y no copiar/pegar del PDF.
*   **Comandos:** Los comandos de terminal se mostrarán en cuadros con el prefijo '$'.
*   **Copyright y Descargo de Responsabilidad:** Todos los documentos incluirán un texto de protección de derechos de autor y un descargo de responsabilidad, indicando que los materiales son para uso académico exclusivo y no pueden ser redistribuidos sin permiso explícito.
